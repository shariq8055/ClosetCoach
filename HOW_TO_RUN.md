# How to Run ClosetCoach

## Quick Start (Two Terminals Required)

### Terminal 1: Frontend
```bash
cd Final-AI/frontend
npm run dev
```
Opens at: http://localhost:5173

### Terminal 2: Backend  
```bash
cd ClosetCoach-Backend
python api.py
```
Runs on: http://localhost:5050

## In VS Code Steps:

1. Press `Ctrl+`` to open terminal
2. Run the frontend command above
3. Click the `+` button to create second terminal
4. Run the backend command
5. Open http://localhost:5173 in browser
